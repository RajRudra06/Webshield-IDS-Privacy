WebShield IDS is a browser extension designed to help users identify potentially malicious or phishing websites while browsing.

Data Collection

WebShield IDS does not collect, sell, or share personally identifiable information.

The extension may process the following data locally or transiently for security analysis:

Visited website URLs

Page metadata required for threat detection

Extension configuration preferences

Data Usage

URLs are analyzed to detect phishing or malicious patterns.

User settings are stored locally using Chrome storage.

Optional backend communication may occur strictly for threat intelligence and model inference.

No browsing history is stored permanently.

No data is sold or shared with third parties.

Permissions Justification

Tabs & Host permissions: Required to analyze the active webpage.

webRequest: Used to detect malicious redirects.

Storage: Used to save user preferences.

Notifications: Used to alert users of detected threats.

Scripting: Used to analyze webpage content for security risks.

Data Security

All processing follows the Chrome Web Store Developer Program Policies. No remote executable code is downloaded or run.
